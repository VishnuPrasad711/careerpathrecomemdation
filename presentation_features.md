# AI Career Coach - Page Features overview

## 1. Landing Page (`landing.html`)
- **Hero Section:** Introduces the platform, inviting users to elevate their future with intelligent career coaching.
- **Feature Highlights:** Showcases core offerings including instant resume analysis, personal AI coaching, and daily challenge streaks.
- **Quick Access:** Provides immediate calls-to-action for users to easily log in or sign up.

## 2. Authentication Page (`index.html`)
- **Secure Access:** The main gateway where users can seamlessly log in or toggle to create a new account.
- **Smart Redirection:** Automatically detects active sessions and routes logged-in users directly to their dashboard.

## 3. Main Dashboard (`dashboard.html`)
- **Profile Management:** Users can manually input their skills and experience, or auto-fill them via PDF resume upload.
- **AI Analysis & Alignment:** Generates professional insights, matches top career roles, and identifies critical skill gaps.
- **Visual Analytics:** Features dynamic charts visualizing role match percentages and current job market demand.
- **Learning Pathways:** Suggests personalized courses and learning steps based on the user's career goals.
- **Recruiter Intel:** Lists specific companies and recruiters actively hiring for the user's matched roles.
- **ATS Scanner:** Compares user resumes against pasted job descriptions to reveal match scores, keyword hits, and missing skills.
- **AI Interview Coach:** A real-time conversational chatbot dedicated to answering job-hunt queries and prepping for interviews.
- **Gamification:** Keeps users engaged with login streaks and mini-games (like developer trivia and speed typing) to earn points.

## 4. Personalized Recommendations (`recommendations.html`)
- **Focused Career Path:** A dedicated, standalone page that expands on the user's AI-generated career recommendations.
- **Deep Dive Insights:** Organizes the Analysis, Charts, Courses, and Companies into a clean, presentation-ready layout.
